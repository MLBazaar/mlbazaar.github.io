FAQs

How can I learn more about ML Bazaar?

We are releasing a paper soon. It will describe primitives, pipelines and primitives. Stay tuned and we will update here.

What open source libraries are you releasing?

We are releasing a primitive library - MLPrimitives, a pipeline execution library - MLBlocks, and a collection of techniques typically used for AutoML in a library called BTB.

How can I programmatically explore datasets, pipelines, templates?

You can programmatically access the datasets (organized as D3M format) and about 3.1 million pipelines via Pipeline explorer library. We are adding millions more.

How can I contribute?

We will be opening up ability to contribute to all our libraries. You would be able to contribute a primitive, suggest a pipeline, suggest hyperparameters or their search ranges. Once you follow the steps and the code is validated, we will run the experiments and return results. If your pipeline does better, we will update the website here and credit you/your team with it.

Is this just an academic exercise?

We are motivated to provide a place where any practitioner can go and find validated competitive templates for their machine learning task.
